Infrastructure Summary

  • Proxmox cluster with 3 nodes: kiwi, pve2, pve3
  • Datastores: nvme (VM disks), iso-storage (cloud images)
  • OPNsense2 firewall (VM 109) with dual NIC for network segmentation
  • K3S cluster: 1 server + 2 agent nodes, MetalLB LB, NFS persistent storage, NGINX Proxy Manager
  • Active workload VMs: Hermes (AI agent), Jellyfin (media streaming)
  • Deprecated: Hytale (game server), CORS-Proxy (VM 106), Ubuntu-Server-1
  • Shared vm_ubuntu module: Ubuntu Noble cloud-init, SSH key auth, qemu-guest-agent, x86-64-v2-AES CPU
  • All VMs provisioned via bpg/proxmox provider with environment-variable credentials
